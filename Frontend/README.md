# fun41-vue-frontend

### Docker startup

```
building image:
docker build -t test .

run image:
docker run -it -p 8082:80 --rm --name test2 test

building with docker compose:
docker-compose up -d --build

running application with docker compose:
docker-compose up
```

npm install
